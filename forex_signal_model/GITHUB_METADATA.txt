🚀 **Forex Signal Model** — Production-Ready ML Trading System

An enterprise-grade machine learning framework for generating trading signals on forex and cryptocurrency pairs. Built with XGBoost, integrated with Binance & Yahoo Finance APIs, and featuring a real-time Flask dashboard for signal monitoring.

**Key Features:**
✨ XGBoost-based predictive model for 5m-1d scalp & spot trading
📊 Multi-source data: Binance REST API + Yahoo Finance
🎯 Real-time signal generation with entry/exit levels
📈 Full backtesting pipeline for strategy validation
🖥️ Interactive Flask web dashboard with OHLCV charts
🔌 MetaTrader 4/5 integration layer (extensible)
⚙️ Modular architecture for easy customization
📦 Production-ready error handling & logging

**Tech Stack:**
- Python 3.8+
- XGBoost, scikit-learn, pandas, numpy
- Flask + Flask-CORS (web server)
- Binance API, yfinance (data sources)
- SHAP (model interpretability)

**Quick Start:**
```bash
pip install -r requirements.txt
python scripts/train.py --symbol BTCUSDT --interval 5m
python app.py  # Dashboard at http://localhost:5000
```
